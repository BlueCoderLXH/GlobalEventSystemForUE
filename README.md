# GlobalEventSystemForUE

## 概述
GES(GlobalEventSystem)，基于UE(当前版本4.26)开发的全局事件系统，功能特性：
* 实现三端数据互通（C++/Lua/Blueprint）
* 事件数据配置热加载
* 轻量
* 易扩展
* 易使用
